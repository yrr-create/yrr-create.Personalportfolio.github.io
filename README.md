# Yi Rongrong Portfolio

Static personal portfolio for GitHub Pages.

## Planned structure

- `index.html` - homepage
- `styles/` - CSS files
- `scripts/` - JavaScript files
- `data/` - project data
- `assets/images/` - profile and project images
- `assets/videos/` - small demo videos
- `assets/docs/resume/` - public resume PDF
- `projects/` - project detail pages
